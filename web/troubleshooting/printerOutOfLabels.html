Follow these steps to replace the roll of labels in the printer

Instructions for the Brother QL580N

Go into printing preferences and set format to 'normal format', width to '62mm' and length to '100m'

Instructions for the Brother QL820NWB

Go into printing preferences and set paper size to '62mm' and length to '100m'

Instructions for the DYMO LabelWriter 450Turbo

Go into printing preferences > advanced and set paper size to '99014 Name Badge Label'
